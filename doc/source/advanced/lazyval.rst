遅延評価
------------------------------------

遅延評価とは、変数の初期化のタイミングを、その変数が初めて利用されるときまで遅延させる仕組みです。

遅延評価(lazy val)の利用
__________________________________

あるクラスFooの定義が、以下のようになっていたとしましょう。


.. code-block:: scala

  class Foo(
    val s = {
      println("sが評価")
      "aaa"
    }
  }
