Scalaの基本
================================================

ここでは、Scalaの基本的な文法や関数の使い方、クラスやオブジェクトの定義、パターンマッチなどを取り上げます。

これから説明する内容は、ぜひScalaインタプリタでサンプルコードを入力して動作を確認しながら読んでみてください。

なお、文中に「したほうがよい。」「が推奨される。」などの記述がありますが、一部著者の私見が混ざっている可能性があることをご了承ください。
